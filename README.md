# ⚡ Apex Renderer: Vector Engine
**The ultimate high-performance rendering overhaul by ExodusCoder9.**

Apex Renderer is a sophisticated UI and performance reconstruction for Minecraft 1.21.1. Built on the **Java 26 Vector API**, it utilizes AVX2/AVX-512 instruction sets to process vertex math in parallel.

### 💎 Features
* **SIMD Vector Core:** Massive parallel math processing for chunk rendering.
* **Indirect Draw:** Optimized GPU-side batching.

### 📥 Requirements
Requires **Java 26** , Fabric Loader 1.21.11, and the `--add-modules jdk.incubator.vector --enable-preview` JVM flags.
